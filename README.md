# 📚 TextEditor (WPF)

Учебный проект на WPF — текстовый редактор с расширенной функциональностью для работы со словарём иностранного языка.

---

## 🎯 Цель проекта

- углубление знаний C# и WPF
- изучение XAML
- создание практического приложения для личного использования
- работа с базой данных и структурированными данными

---

## ⚙️ Основной функционал

### ✏️ Текстовый редактор
- изменение шрифта
- настройка размера текста
- выбор типа шрифта
- изменение цвета текста
- базовое редактирование текста

---

### 📖 Режимы работы
- режим редактирования
- режим чтения
- переключение между режимами

---

### 📁 Работа с файлами
- импорт текста
- экспорт текста
- сохранение данных

---

### 📚 Словарь
- работа с иностранными словами (английский язык)
- использование готовых словарей (конвертированных в формат проекта)
- добавление и хранение слов
- чтение словаря внутри приложения

---

### 💾 Хранение данных
- база данных SQLite
- сохранение словаря и пользовательских данных

---

### 🧠 Режим тренировки
- проверка знаний слов
- тестирование запоминания
- повторение и закрепление словарного запаса

---

## 🧰 Технологии

- C#
- WPF
- XAML
- SQLite

---

## 📌 Итог

Приложение объединяет текстовый редактор и систему изучения словаря, позволяя одновременно работать с текстом и изучать иностранные слова в удобной форме.

---

## 👨‍💻 Автор

Учебный проект для практики WPF и разработки прикладных приложений на C#.
